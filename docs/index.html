Welcome to the HIRES PRV Reduction Service Documentation¶


The HIRES PRV pipeline can be used to reduce radial velocity data from the Keck HIRES instrument. We provide access to the Keck Online Archive (KOA) data and to that pipeline as a web service.


These Python functions provide an easy way to interact with the service. The source code is available here.
